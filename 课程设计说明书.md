# 学生信息管理系统 - 课程设计说明书

## 一、需求分析

### 1.1 系统背景
随着教育信息化的发展,学生信息管理成为学校日常管理的重要组成部分。本系统旨在通过计算机技术实现学生信息的电子化管理,提高管理效率,减少人工操作错误。

### 1.2 系统目标
- 实现学生基本信息的录入、修改、删除和查询
- 提供多种搜索方式,快速定位学生信息
- 实现数据的持久化存储
- 提供数据统计分析功能
- 界面友好,操作简便

### 1.3 功能需求
1. **学生信息管理**
   - 添加学生信息
   - 修改学生信息
   - 删除学生信息
   - 查看学生列表

2. **搜索功能**
   - 按学号搜索
   - 按姓名搜索
   - 按专业搜索
   - 按班级搜索
   - 综合搜索

3. **数据管理**
   - 数据持久化存储
   - 数据备份
   - 数据排序

4. **统计分析**
   - 学生人数统计
   - 性别分布统计
   - 平均年龄计算
   - 专业分布统计

### 1.4 性能需求
- 系统响应时间小于1秒
- 支持至少1000条学生记录
- 数据保存成功率100%
- 界面友好,易于操作

## 二、系统设计

### 2.1 系统架构
本系统采用MVC(Model-View-Controller)分层架构:
- **Model层**: 数据模型(Student类)
- **View层**: 用户界面(GUI包)
- **Controller层**: 业务逻辑(Service包)

### 2.2 模块划分

#### 2.2.1 实体模块(model包)
- `Student.java`: 学生实体类,封装学生的基本属性

#### 2.2.2 服务模块(service包)
- `FileManager.java`: 文件管理类,负责数据的读写操作
- `StudentManager.java`: 学生管理类,负责业务逻辑处理

#### 2.2.3 界面模块(gui包)
- `MainFrame.java`: 主窗口,显示学生列表和提供操作按钮
- `StudentDialog.java`: 学生信息编辑对话框

### 2.3 数据结构设计

#### 2.3.1 Student类
```java
class Student {
    String studentId;      // 学号(主键)
    String name;           // 姓名
    String gender;         // 性别
    int age;              // 年龄
    String major;         // 专业
    String classNumber;   // 班级
    String phoneNumber;   // 联系电话
}
```

#### 2.3.2 数据存储
使用ArrayList<Student>存储学生对象,提供高效的增删改查操作。

### 2.4 文件格式设计
采用CSV格式存储数据:
```
学号,姓名,性别,年龄,专业,班级,联系电话
2021001,张三,男,20,计算机科学与技术,计算机2101,13800138001
```

### 2.5 界面设计

#### 2.5.1 主窗口
- 顶部:搜索栏和工具栏
- 中间:学生信息表格
- 底部:状态栏
- 菜单:文件、编辑、排序、帮助

#### 2.5.2 编辑对话框
- 学号输入框(添加时可编辑,修改时禁用)
- 姓名输入框
- 性别下拉框
- 年龄选择器
- 专业输入框
- 班级输入框
- 电话输入框

## 三、详细设计

### 3.1 核心算法

#### 3.1.1 搜索算法
使用Java Stream API进行过滤:
```java
students.stream()
    .filter(s -> s.getName().contains(keyword))
    .collect(Collectors.toList());
```

#### 3.1.2 排序算法
使用Comparator进行排序:
```java
students.sort(Comparator.comparing(Student::getStudentId));
```

#### 3.1.3 统计算法
使用Stream API的groupingBy和counting:
```java
Map<String, Long> majorDistribution = students.stream()
    .collect(Collectors.groupingBy(Student::getMajor, Collectors.counting()));
```

### 3.2 异常处理

#### 3.2.1 文件操作异常
- FileNotFoundException: 文件不存在时的处理
- IOException: 读写错误时的处理
- 使用try-with-resources确保资源正确关闭

#### 3.2.2 数据验证异常
- IllegalArgumentException: 数据格式错误
- NumberFormatException: 数字格式错误

#### 3.2.3 业务逻辑异常
- 学号重复检查
- 必填字段验证
- 数据格式验证

### 3.3 关键代码实现

#### 3.3.1 数据持久化
```java
public boolean saveStudents(List<Student> students) {
    try (BufferedWriter writer = new BufferedWriter(
            new OutputStreamWriter(new FileOutputStream(filePath), "UTF-8"))) {
        for (Student student : students) {
            writer.write(student.toCSV());
            writer.newLine();
        }
        return true;
    } catch (IOException e) {
        e.printStackTrace();
        return false;
    }
}
```
