## 使用方法

### 方法一:使用脚本(推荐)

#### Windows系统:
1. 双击 compile.bat 编译项目
2. 双击 run.bat 运行项目

#### Linux/Mac系统:
1. 打开终端,进入项目目录
2. 执行: ./compile.sh
3. 执行: ./run.sh

### 方法二:手动编译运行

#### 编译:
```bash
cd StudentManagementSystem/src
javac -encoding UTF-8 Main.java model/*.java service/*.java gui/*.java
```

#### 运行:
```bash
cd StudentManagementSystem/src
java Main
```

### 方法三:使用IDE

#### IntelliJ IDEA:
1. File -> Open -> 选择 StudentManagementSystem 目录
2. 右键点击 src 目录 -> Mark Directory as -> Sources Root
3. 右键点击 Main.java -> Run 'Main.main()'

#### Eclipse:
1. File -> Open Projects from File System -> 选择 StudentManagementSystem 目录
2. 右键点击项目 -> Build Project
3. 右键点击 Main.java -> Run As -> Java Application

## 常见问题

Q: 编译提示"javac: 未找到命令"?
A: 需要安装JDK并配置环境变量。

Q: 运行时中文乱码?
A: 确保使用UTF-8编码编译: javac -encoding UTF-8

Q: 数据文件在哪里?
A: 在 StudentManagementSystem/data/students.txt

Q: 如何备份数据?
A: 点击"备份数据"按钮,会在data目录生成students.txt.backup
