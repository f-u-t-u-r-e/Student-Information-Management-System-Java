# 学生信息管理系统

## 项目简介

这是一个基于Java Swing开发的大学生信息管理系统,提供了完整的学生信息增删改查、成绩管理、专业排名、搜索、统计和数据持久化功能。

## 功能特点

### 1. 学生信息管理
- **添加学生**: 录入新学生的完整信息(学号、姓名、性别、年龄、专业、班级、联系电话)
- **编辑学生**: 修改已有学生的信息(学号不可修改)
- **删除学生**: 删除指定学生记录,包含确认提示
- **查看学生**: 表格形式展示所有学生信息

### 2. 成绩管理(新增功能)
- **灵活的课程管理**: 每个学生的课程数量和学分不固定
- **成绩导入**: 支持从CSV文件批量导入学生成绩
- **加权平均分**: 根据课程学分自动计算GPA(加权平均分)
- **课程信息**: 每门课程包含课程名、学分和成绩

### 3. 专业排名(新增功能)
- **按专业查看**: 选择不同专业查看该专业学生排名
- **GPA排序**: 根据加权平均分进行排名
- **排名统计**: 显示专业人数、平均GPA等统计信息
- **排名导出**: 可导出专业排名数据

### 4. 搜索功能
- **多条件搜索**: 支持按学号、姓名、专业、班级进行精确或模糊搜索
- **综合搜索**: 在全部字段中进行关键字搜索
- **实时搜索**: 输入关键字后即时显示搜索结果

### 5. 数据排序
- 按学号排序
- 按姓名排序
- 按年龄排序
- 按GPA排序(新增)

### 6. 统计分析
- 学生总人数统计
- 男女生人数统计
- 平均年龄计算
- 专业分布统计

### 7. 数据管理
- **文件持久化**: 使用CSV格式存储数据到文本文件
- **数据备份**: 一键备份当前数据
- **自动保存**: 每次数据变更自动保存到文件

## 技术特点

### 符合课程要求

#### 1. GUI界面 (第九章)
- 使用Java Swing构建图形用户界面
- 包含JFrame、JPanel、JTable、JDialog等组件
- 实现了菜单栏、工具栏、表格、对话框等完整界面元素

#### 2. 文件I/O (第十章)
- 使用BufferedReader/BufferedWriter进行文件读写
- 实现CSV格式的数据序列化和反序列化
- 支持UTF-8编码,避免中文乱码
- 包含文件备份功能

#### 3. Java集合框架 (第十五章)
- 使用ArrayList存储学生对象
- 使用HashMap进行统计数据存储
- 使用Stream API进行数据过滤和统计
- 使用Comparator进行数据排序

#### 4. 异常处理 (第七章)
- 对文件操作进行完整的异常处理
- 对用户输入进行验证和异常捕获
- 提供友好的错误提示信息
- 使用try-catch-finally和try-with-resources

## 使用说明

### 1. 启动系统
运行Main.java,系统会自动加载data/students.txt中的数据

### 2. 添加学生
- 点击"添加学生"按钮
- 在弹出的对话框中填写学生信息
- 点击"确定"保存

### 3. 编辑学生
- 在表格中选择要编辑的学生
- 点击"编辑学生"按钮或双击该行
- 修改信息后点击"确定"

### 4. 删除学生
- 在表格中选择要删除的学生
- 点击"删除学生"按钮
- 确认删除操作

### 5. 搜索学生
- 选择搜索类型(学号/姓名/专业/班级/全部)
- 输入搜索关键字
- 点击"搜索"按钮或按回车键

### 6. 查看统计
- 点击"统计信息"按钮
- 查看学生人数、性别分布、平均年龄和专业分布

### 7. 数据备份
- 点击"备份数据"按钮
- 系统会在data目录下创建students.txt.backup文件

### 8. 导入成绩(新增功能)
- 点击"导入成绩"按钮
- 选择成绩CSV文件(可参考data/scores_import_example.csv)
- 系统自动导入并计算加权平均分
- 成绩文件格式: 学号,课程名,学分,成绩

### 9. 查看专业排名(新增功能)
- 点击"专业排名"按钮
- 从下拉框选择要查看的专业
- 查看该专业学生的GPA排名
- 可导出排名数据

## 数据格式

### 学生数据文件格式

采用CSV格式,支持课程成绩信息:

```
学号,姓名,性别,年龄,专业,班级,联系电话,[课程1:学分:成绩|课程2:学分:成绩|...]
2021001,张三,男,20,计算机科学与技术,计算机2101,13800138001,[高等数学:4.0:85.5|大学英语:3.0:90.0|程序设计基础:4.0:92.0]
```

### 成绩导入文件格式

CSV格式,每行一条成绩记录:

```
学号,课程名,学分,成绩
2021001,高等数学,4.0,85.5
2021001,大学英语,3.0,90.0
2021002,高等数学,4.0,91.0
```

## 输入验证

- **学号**: 必填,添加后不可修改
- **姓名**: 必填
- **性别**: 下拉选择(男/女)
- **年龄**: 数字选择器,范围15-100
- **专业**: 必填
- **班级**: 必填
- **联系电话**: 必填,必须为11位数字

## 注意事项

1. 确保data目录存在,系统会自动创建数据文件
2. 数据文件采用UTF-8编码,确保中文正常显示
3. 学号作为唯一标识,不允许重复
4. 删除操作不可恢复,请谨慎操作
5. 建议定期使用备份功能保存数据
6. 成绩导入时,确保学号已存在于系统中
7. GPA计算公式: GPA = Σ(课程成绩×学分) / Σ学分
